# DigiBalance — Digital Wellness Tracker

This version removes the chatbot and all AI-server dependencies so the project installs and starts faster.

## Run locally

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run lint
npm run build
```

## Included features

- Digital wellness dashboard
- Screen-time and app-usage tracking
- Focus timer
- Digital detox planner
- Mood, energy, sleep, and break check-ins
- WHO-inspired public-health awareness
- Weather-aware break guidance
- Analytics, profile settings, themes, moving headline, and guided page instructions

The original DigiBalance logo and the generated wellness images remain unchanged.
